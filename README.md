# Auto Sender Baggage Bot 📦

Bot នេះប្រើសម្រាប់ទាញយកលេខទូរស័ព្ទពីលើប្រអប់ឥវ៉ាន់តាមរយៈរូបភាព (ប្រើ EasyOCR) បន្ទាប់មកផ្ញើសារទៅកាន់អតិថិជនដើម្បីផ្ទៀងផ្ទាត់លេខទូរស័ព្ទ និងសុំទីតាំងសម្រាប់ការដឹកជញ្ជូន។

## របៀបដំឡើង (How to setup)
1. Clone repository នេះ
2. ដំឡើងបណ្ណាល័យតាមរយៈពាក្យបញ្ជា៖ `pip install -r requirements.txt`
3. បង្កើតឯកសារ `.env` និងបញ្ចូល `BOT_TOKEN`, `BOT_USERNAME`, `GROUP_CHAT_ID`
4. ដំណើរការ Bot តាមរយៈពាក្យបញ្ជា៖ `python bot.py`
